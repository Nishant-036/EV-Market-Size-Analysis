# EV Market Size Analysis using Python

## Project Overview
This project analyzes the Electric Vehicle (EV) market using Python data analysis and visualization techniques.

The project includes:
- Data Cleaning
- Exploratory Data Analysis (EDA)
- Visualization
- Correlation Analysis
- EV Growth Trend Analysis
- Forecasting Future EV Registrations

---

## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn

---

## Features of the Project
- EV registrations by model year
- Top EV manufacturers
- Top EV models
- EV type distribution
- Electric range analysis
- Correlation matrix
- City and county-wise EV analysis
- Forecasting future EV growth

---

## Key Findings
- EV adoption increased rapidly after 2015.
- Tesla dominates the EV market.
- Battery Electric Vehicles (BEVs) have the highest market share.
- Urban regions show higher EV registrations.
- Future EV registrations are expected to increase steadily.
